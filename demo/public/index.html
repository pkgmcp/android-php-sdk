📖 How It Works

Step-by-step guide to using the demo panel with your Android device.

Prerequisites

1
Enable Developer Options
Go to Settings → About phone → Tap "Build number" 7 times.
2
Enable USB Debugging
Settings → Developer options → USB debugging → Enable.
3
Start ADB Server
adb start-server on your computer. Connect device via USB.
4
Authorize the Connection
On your device, tap "Allow" on the USB debugging prompt.

Using Each Section

📱 Device List

See all connected devices, their state (device/offline/unauthorized), and USB paths.

ℹ️ Device Info

Get detailed info: model, Android version, SDK, CPU ABI, IP address, manufacturer.

💻 Shell

Run any shell command on the device: uname -a, ls /sdcard, getprop, etc.

📦 Install / Uninstall

Install APK files, uninstall packages, clear app data, check if a package is installed.

📁 File Manager

Browse device files, push files to device, pull files from device, stat files and directories.

🖥️ Screen

Take screenshots and view them directly in the browser. Get framebuffer metadata.

🔀 Port Forward

Create and list port forwards and reverse forwards between host and device.

⚡ Fastboot
